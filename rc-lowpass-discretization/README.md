# Discretization of an RC Lowpass Filter
Latest .pdf version: http://www.rossbencina.com/static/writings/dsp-notes/rc-lowpass-discretization.pdf

Runs through the steps required to take an analog RC lowpass filtre circuit and turn it into digital difference equations using trapezoidal integration. Based on Andy Simper's technical papers at http://www.cytomic.com/technical-papers
